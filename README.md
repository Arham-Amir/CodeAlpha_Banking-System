C++ Banking System
This is a console-based banking system implemented in C++ using object-oriented programming (OOP) concepts. The project includes classes for Customer, Account, and Transaction, enabling basic banking operations such as deposits, withdrawals, fund transfers, and viewing transaction history.

Each Customer has a unique account and can perform operations on their Account. The system tracks transactions using the Transaction class, which records deposits, withdrawals, and transfers. Users can view account information including balance and past transactions.

Key Features:

Customer Management: Create and manage customer accounts with unique account numbers.
Deposit & Withdrawal: Securely add or withdraw funds from accounts.
Fund Transfers: Transfer money between accounts while updating transaction history.
Transaction History: Keep track of all account activities for transparency.
Account Information: Display account holder details and current balance.

This project demonstrates practical usage of OOP principles in C++, including class design, encapsulation, object interactions, and vector-based dynamic storage. It is beginner-friendly and can be extended to include persistent file storage, GUI, or integration with databases for real-world applications.

Technologies Used:

C++
Classes and Objects
Vectors for dynamic transaction storage
Console-based interface
